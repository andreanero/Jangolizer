KEY FEATURES
════════════════════════════════════════════════════════════════════════════
✅ PolyBLEP Anti-Aliased LFO (0.1 Hz - 400 Hz)
✅ Dual Processing Modes:
   • VCA Mode: Tremolo & Ring Modulation
   • VCF Mode: Dynamic Bandpass Filter
✅ 6 Real-Time Parameters with Automation
✅ Full State Serialization (Preset Save/Load)
✅ Dual Build Targets:
   • Desktop GUI (Ardour/Linux)
   • Headless (Elk Audio OS on Raspberry Pi)
✅ Real-Time Safe (Zero Allocations in Audio Loop)
✅ Cross-Platform (macOS, Windows, Linux)

QUICK START
════════════════════════════════════════════════════════════════════════════

Test Standalone:
  $ cd /home/andrea/Documents/juce_course/jangolizer
  $ ./cmake-build/GristleizerPlugin_artefacts/Debug/Standalone/Gristleizer

Install VST3 for Ardour:
  $ mkdir -p ~/.vst3
  $ cp -r cmake-build/GristleizerPlugin_artefacts/Debug/VST3/Gristleizer.vst3 ~/.vst3/

Install LV2 for Ardour:
  $ mkdir -p ~/.lv2
  $ cp -r cmake-build/GristleizerPlugin_artefacts/Debug/LV2/Gristleizer.lv2 ~/.lv2/

Release Build (Optimized):
  $ ./build.sh release

Elk Audio OS (Headless):
  $ ./build.sh elk
